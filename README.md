## 📌 Overview  
SafePulse is a **smart personal safety and emergency response system** designed to provide:
- 🚨 Instant SOS alerts  
- 📍 Real-time location tracking  
- 🧠 Intelligent escalation system  

👉 It ensures help reaches you **even when your contacts don’t respond**.

---

## 🎯 Key Highlight (🔥 Unique Feature)

### 🧠 Smart Escalation System  
If no one responds:
➡️ Alert is sent to nearby users  
➡️ Location is shared locally  
➡️ Community members can help instantly  

💡 *Because safety should never depend on just one person.*
